Project Roadmap

  • 2016-2017: Experimentation

    Founder Kevin Owocki experiments with Ethereum mining, building a handful of Open Source projects in the space, investing, and community organizing.

  • Q2 2017: Ideation

    With the mentorship of Open Source Developers from the Boulder CO tech community, the idea of a decentralized network designed to Push Open Source Repos Forward is conceived. What if recruitment for software projects could be completely decentralized and automated? What if developers could try projects before they commit? What if repo maintainers had a way to bring new developers onto their projects?

  • Q3 2017: Build & Design

    Customer Development commences. The founder codes. The design team designs. Animator animates.

  • Q4 2017: Launch

    The Issues Explorer goes live. The code behind the project is released open source. For the first time, coders can monetize their open-source work without a centralized middleman, and an incentive is provided to the open-source community to Push Open Source Repos Forward, together.

  • Next: Tools++

    Funded Issues are processed through the system, and feedback is processed into building a better, faster, cheaper way of pushing open source forward. The system is dogfooded; Gitcoin tools used to build Gitcoin.

  • After that: Growth

    The market for software jobs is massive, and Open Source software creates billions of dollars in economic value per year. What will the world look like if decentralized ledger technology begins to disrupt the existing value systems in software?
